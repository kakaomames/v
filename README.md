## ✅ ワンポイントアドバイス

- **動画＋音声を一発で取得したい** → **itag: 18 (360p)** や **22 (720p)**  
- **音声だけ取得したい** → **itag: 140 (M4A 128kbps)**  
- **最高画質を狙いたい** → **itag: 313 (4K)** と **140 (音声)** を組み合わせる
